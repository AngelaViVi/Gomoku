# UnityGomoku

使用Unity 3D编写的人机五子棋程序
===========================

### 开发环境
<br>Visual Studio 2015 or higher
<br>Unity 3D 2017.2.0f3

### AI算法
<br>α-β剪枝
<br>置换表,历史值启发,迭代增量
